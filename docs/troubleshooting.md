# Troubleshooting

Common issues and their solutions.

## Installation Issues

### Issue: Plugin not loading

**Solution:** Ensure the plugin is listed in your `opencode.json` file.

## Usage Issues

### Issue: Feature not working

**Solution:** Check your configuration and ensure all required options are set.

## Performance Issues

### Issue: Plugin is slow

**Solution:** Review your configuration and consider optimizing any custom implementations.

## Getting Help

If you encounter an issue not listed here:

1. Check the [GitHub Issues](https://github.com/pantheon-org/opencode-warcraft-notifications-plugin/issues)
2. Search for similar issues
3. Create a new issue if needed

Include the following information:

- OpenCode version
- Plugin version
- Operating system
- Configuration file
- Error messages
